Single-page web tool for GitHub Pages

Project Overview

This artifact grew out of my Workshop 1 leadership reflection work. Rather than only posting a written reflection, I wanted to turn that experience into something more useful for others: a simple tool that lets a visitor evaluate themselves in AI/ML leadership and change management, view visual results, and download a report.

This makes the artifact more than a class assignment. It becomes an example of how reflective coursework can be transformed into an audience-facing professional resource.

What the Tool Includes

  • Two self-assessment tracks: AI/ML leadership and change management
  • 1-to-5 rating scale for each statement
  • Automatic category averages
  • Overall visual score display
  • Downloadable report for reflection or follow-up use

Why It Matters

  • Shows initiative beyond a standard written reflection
  • Demonstrates audience-oriented design thinking
  • Highlights leadership development as an ongoing process
  • Adds interactivity to the portfolio instead of static text alone

Reflection

I see this artifact as a bridge between self-assessment and public-facing professional communication. It reflects the idea that leadership growth can be made visible, measurable, and useful to others. In the future, this concept could be expanded with more polished graphics, more nuanced reports, or additional reflection prompts, but the current version already shows the core value of the idea.

Professional Signal

This artifact signals that I can turn internal learning and reflection into a tool that serves a broader audience, which is valuable in educational, professional, and product-oriented settings.

Current Scope

The portfolio version is intentionally lightweight for GitHub Pages, but it already demonstrates concept design, structure, scoring logic, and visual reporting.
